## Commands

```bash
# Development
npm run dev                      # Dev server
npm run build                    # Build for production
npm run preview                  # Preview built output

# Quality
npm run lint                     # Lint (if configured)
npm run typecheck                # Type check (if TypeScript)

# Deployment
npm run deploy                   # Deploy (project-specific)
```

## Quick Reference

- **Before changes**: Understand the content structure
- **During changes**: Preview in browser, check responsiveness
- **After changes**: Build succeeds + user approves visually
- **Commits**: Describe content/design changes
